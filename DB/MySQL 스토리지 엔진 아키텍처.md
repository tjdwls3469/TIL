### MySQL (MySQL 서버)

* MySQL 엔진과 스토리지 엔진을 모두 합친걸 의미한다.
* Mysql 엔진은 하나만 사용할 수 있지만 스토리지 엔진은 동시에 여러 개를 사용할 수 있다.
* 애플리케이션 프로그래머와 DBA를 분리하여 일관되고 쉬운 애플리케이션 모델 및 API를 제공한다.
